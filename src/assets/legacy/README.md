# Legacy Files

이 폴더에는 React로 전환하기 전의 기존 파일들이 보관되어 있습니다.

## 파일 목록

- `styles.css` - 기존 전체 스타일시트 (현재는 컴포넌트별로 분리됨)
- `app.js` - 기존 메인 JavaScript 로직 (현재는 React 컴포넌트로 변환됨)
- `start.js` - 기존 시작 화면 JavaScript (현재는 `StartScreen.jsx`로 변환됨)
- `main.html` - 기존 HTML 구조 (현재는 React 컴포넌트로 변환됨)

## 참고사항

이 파일들은 참고용으로만 보관되어 있으며, 현재 프로젝트에서는 사용되지 않습니다.
React 컴포넌트 구조를 참고하세요.
